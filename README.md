# Painel de Comunicados Internos

Solução da **Questão 3** do case técnico — vaga de estágio de Canais e Dados / Comunicação (BUPJ).

## Sobre o projeto

Painel simples que simula uma publicação de comunicados internos, pensado para ser integrado a uma intranet corporativa (ex: SharePoint). Ao clicar no botão, a lista de comunicados é exibida ou ocultada, sem recarregar a página.

## Tecnologias utilizadas

- **HTML5** — estrutura da página
- **CSS3** — estilo visual
- **JavaScript (vanilla)** — comportamento interativo

## Estrutura dos arquivos

​├── index.html      # estrutura e conteúdo

├── style.css       # aparência

└── script.js       # comportamento (clique, exibir/ocultar)
